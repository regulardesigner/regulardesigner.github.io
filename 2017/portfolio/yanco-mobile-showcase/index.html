Showcase

Une application mobile de présentation de l'inventaire publicitaire pour Yanco Mobile.

Présenter les formats publicitaires mobiles dans leurs contextes naturels aussi simplement que sortir son smartphone de sa poche.

 
Le client :

Fondé en 2012, Yanco Mobile est une société de vente d’espaces publicitaires issus du programmatique (R.T.B. Real Time Bidding).

 
Le projet :

Création d’une application mobile de présentation des formats publicitaires mobile.

 
Les utilisateurs :

Les commerciaux de Yanco Mobile. CSP +, 25/35 ans.

 
Objectifs :

Présentation « On the go » et dans un contexte réel des formats publicitaire innovant de l’inventaire de Yanco Mobile. Supprimer le temps alloué à la création d’une présentation classique (PPT) pour présenter ces formats aux clients.

 
Process & outils :

Users interview, Wireframes , Design d'interface.
